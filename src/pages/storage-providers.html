Secure and lightweight

The CIDgravity connector is an unprivileged open-source Python script that utilizes the standard dealFilter mechanism from Boost and Venus.

The logic is straightforward: each time the miner receives a deal, it is first sent to CIDgravity APIs to determine if it should be accepted based on the settings you've defined.
